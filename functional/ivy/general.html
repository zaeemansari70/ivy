ivy.fourier_encode(x: Union[ivy.array.Array, ivy.NativeArray], max_freq: Union[float, ivy.array.Array, ivy.NativeArray], num_bands: int = 4, linear: bool = False, concat: bool = True, flatten: bool = False) → Union[ivy.array.Array, ivy.NativeArray, Tuple][source]¶

Pads an array with fourier encodings.

Parameters

  • x (array) – Input array to encode.

  • max_freq (float) – The maximum frequency of the encoding.

  • num_bands (int, optional) – The number of frequency bands for the encoding. Default is 4.

  • linear (bool, optional) – Whether to space the frequency bands linearly as opposed to geometrically. Default is False.

  • concat (bool, optional) – Whether to concatenate the position, sin and cos values, or return seperately. Default is True.

  • flatten (bool, optional) – Whether to flatten the position dimension into the batch dimension. Default is False.

Returns

New array with the final dimension expanded, and the encodings stored in this channel.

ivy.scatter_flat(indices: Union[ivy.array.Array, ivy.NativeArray], updates: Union[ivy.array.Array, ivy.NativeArray], size: Optional[int] = None, tensor: Optional[Union[ivy.array.Array, ivy.NativeArray]] = None, reduction: str = 'sum', dev: Optional[ivy.Device] = None) → Union[ivy.array.Array, ivy.NativeArray][source]¶

Scatter flat updates into a new flat array according to flat indices.

Parameters

  • indices (array) – Indices for the new values to occupy.

  • updates (array) – Values for the new array to hold.

  • size (int) – The size of the result.

  • tensor – The tensor in which to scatter the results, default is None, in which case the size is used to scatter into a zeros array.

  • reduction (str) – The reduction method for the scatter, one of ‘sum’, ‘min’, ‘max’ or ‘replace’

  • dev (ivy.Device, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as updates if None.

Returns

New array of given shape, with the values scattered at the indices.
