ivy.allclose(a, b, /, *, rtol=1e-05, atol=1e-08, equal_nan=False, out=None)[source]ÔÉÅ

Returns a True if the two arrays are element-wise equal within given tolerance; otherwise False. The tolerance values are positive, typically very small numbers. The relative difference (rtol * abs(x2)) and the absolute difference atol are added together to compare against the absolute difference between x1 and x2. The default atol is not appropriate for comparing numbers that are much smaller than one

Parameters

  • x1 ‚Äì First input array.

  • x2 ‚Äì Second input array.

  • rtol (Optional[float]) ‚Äì The relative tolerance parameter. (default: 1e-05)

  • atol (Optional[float]) ‚Äì The absolute tolerance parameter. (default: 1e-08)

  • equal_nan (Optional[bool]) ‚Äì Whether to compare NaN‚Äôs as equal. If True, NaN‚Äôs in x1 will be (default: False) considered equal to NaN‚Äôs in x2 in the output array.

  • out (Optional[Array]) ‚Äì Alternate output array in which to place the result. (default: None) The default is None.

Return type

bool

Returns

ret ‚Äì Returns True if the two arrays are equal within the given tolerance; False otherwise.

Examples

>>> x1 = ivy.array([1e10, 1e-7])
>>> x2 = ivy.array([1.00001e10, 1e-8])
>>> y = ivy.allclose(x1, x2)
>>> print(y)
ivy.array(False)

>>> x1 = ivy.array([1.0, ivy.nan])
>>> x2 = ivy.array([1.0, ivy.nan])
>>> y = ivy.allclose(x1, x2, equal_nan=True)
>>> print(y)
ivy.array(True)

>>> x1 = ivy.array([1e-10, 1e-10])
>>> x2 = ivy.array([1.00001e-10, 1e-10])
>>> y = ivy.allclose(x1, x2, rtol=0.005, atol=0.0)
>>> print(y)
ivy.array(True)

ivy.sinc(x, /, *, out=None)[source]ÔÉÅ

Calculates an implementation-dependent approximation of the principal value of the normalized sinc function, having domain (-infinity, +infinity) and codomain [-0.217234, 1], for each element x_i of the input array x. Each element x_i is assumed to be expressed in radians.

Special cases

For floating-point operands,

  • If x_i is NaN, the result is NaN.

  • If x_i is 0, the result is 1.

  • If x_i is either +infinity or -infinity, the result is NaN.

Parameters

  • x (Union[Array, NativeArray]) ‚Äì input array. Should have a floating-point data type.

  • out (Optional[Array]) ‚Äì optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Return type

Array

Returns

ret ‚Äì an array containing the normalized sinc function of each element in x. The returned array must have a floating-point data type determined by type-promotion.
