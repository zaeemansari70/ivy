dropout1dÔÉÅ

ivy.dropout1d(x, prob, /, *, training=True, data_format='NWC', out=None)[source]ÔÉÅ
Randomly zero out entire channels with probability prob using samples from

a Bernoulli distribution and the remaining channels are scaled by (1/1-prob). In this case, dropout1d performs a channel-wise dropout but assumes a channel is a 1D feature map.

Parameters

  • x (Union[Array, NativeArray]) ‚Äì a 2D or 3D input array. Should have a floating-point data type.

  • prob (float) ‚Äì probability of a channel to be zero-ed.

  • training (bool) ‚Äì controls whether dropout1d is performed during training or ignored (default: True) during testing.

  • data_format (str) ‚Äì ‚ÄúNWC‚Äù or ‚ÄúNCW‚Äù. Defaults to ‚ÄúNWC‚Äù. (default: 'NWC')

  • out (Optional[Array]) ‚Äì optional output array, for writing the result to. (default: None) It must have a shape that the inputs broadcast to.

Return type

Array

Returns

  • ret ‚Äì

    an array with some channels zero-ed and the rest of channels are

    scaled by (1/1-prob).

  • Both the description and the type hints above assumes an array input for simplicity,

  • but this function is nestable, and therefore also accepts ivy.Container

  • instances in place of any of the arguments.

Examples

With ivy.Array input:

>>> x = ivy.array([1, 1, 1]).reshape([1, 1, 3])
>>> y = ivy.dropout1d(x, 0.5)
>>> print(y)
ivy.array([[[2., 0, 2.]]])

>>> x = ivy.array([1, 1, 1]).reshape([1, 1, 3])
>>> y = ivy.dropout1d(x, 1, training=False, data_format="NCW")
>>> print(y)
ivy.array([[[1, 1, 1]]])

With one ivy.Container input: >>> x = ivy.Container(a=ivy.array([100, 200, 300]).reshape([1, 1, 3]), ‚Ä¶ b=ivy.array([400, 500, 600]).reshape([1, 1, 3])) >>> y = ivy.dropout1d(x, 0.5) >>> print(y) { a: ivy.array([[[200., 400., 0.]]]), b: ivy.array([[[0., 0., 0.]]]) }
